# Gym Management System - Installation Guide

## For the Gym Owner

Your Gym Management System desktop application has been successfully built and is ready for installation.

## Installation Options

### Option 1: Easy Installer (Recommended)
1. Navigate to the `out/make/squirrel.windows/x64/` folder
2. Double-click `Gym Management System-1.0.0 Setup.exe`
3. Follow the installation wizard
4. The application will be installed and a desktop shortcut will be created

### Option 2: Portable Version
1. Navigate to the `out/Gym Management System-win32-x64/` folder
2. Copy the entire folder to your desired location
3. Double-click `Gym Management System.exe` to run the application
4. No installation required - you can run it from any location

## Application Features

Your gym management system includes:
- **Member Registration**: Add and manage gym members
- **Barcode Scanner**: Scan member cards for quick check-ins
- **Visit Tracking**: Monitor member visits and attendance
- **Daily Dashboard**: View statistics and daily reports
- **Employee Management**: Manage staff access and roles
- **Data Storage**: All data is stored locally and securely

## System Requirements

- **Operating System**: Windows 10 or Windows 11
- **Architecture**: 64-bit (x64)
- **Memory**: At least 4GB RAM recommended
- **Disk Space**: Approximately 200MB free space

## File Structure

```
out/
├── Gym Management System-win32-x64/           # Portable version
│   └── Gym Management System.exe              # Main application
└── make/
    └── squirrel.windows/
        └── x64/
            └── Gym Management System-1.0.0 Setup.exe  # Installer
```

## Running the Application

1. After installation, launch the application from:
   - Desktop shortcut (if using installer)
   - Start menu: "Gym Management System"
   - Or directly from the executable file

2. The application will open with a login screen
3. Use the appropriate credentials to access the system

## Data Location

- All member data, visits, and statistics are stored locally on your computer
- Data files are located in the application directory
- Regular backups are recommended
